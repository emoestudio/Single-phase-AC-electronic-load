# 功能汇总

1. 板子设计紧凑美观噪音小，高功率密度

2. 全隔离

3. 功率因数0.1~1 容感性可调

4. 输出电压与输入电压相位跟随

5. 输入电流THD低，2%以下

6. **屏幕显示（界面做好看点）**

   1. 有功无功功率显示
   2. 电压电流电阻检测显示

7. 模式

   1. CC 0.2A~2.5A可调，分辨率0.1A
   2. CR 12欧~150欧可调，分辨率0.5欧
   3. **CP 10~70W可调，分辨率0.5W**
   4. **CS 10~70VA可调，分辨率0.5VA**
   5. 不控整流负载模拟（输出大概给个调制比）

8. 保护

   1. 输入过压
   2. 负载丢失
   3. **输入过流**
   4. 自恢复
